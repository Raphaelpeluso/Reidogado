# # Sistema de Caixa para Açougue Reidogado

## Descrição
Este é um sistema desktop de caixa desenvolvido para gerenciar transações e inventário em um açougue. O programa foi projetado para simplificar as operações diárias, incluindo a venda de produtos, o acompanhamento de estoque e a geração de relatórios financeiros básicos.

## Funcionalidades Principais
- **Registro de Vendas:** Registre as vendas de produtos de forma rápida e eficiente.
- **Gestão de Estoque:** Mantenha um controle preciso do estoque de produtos disponíveis.
- **Relatórios Financeiros:** Gere relatórios que fornecem insights sobre o desempenho financeiro do açougue.
